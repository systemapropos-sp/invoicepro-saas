# InvoicePro - SaaS Invoicing Platform

A modern, feature-rich invoicing platform built with React, TypeScript, and Tailwind CSS.

## 🚀 Features

- **Multi-Business Support**: Manage multiple businesses with isolated data
- **Language Switcher**: English and Spanish support with one-click switching
- **Document Management**: Upload documents and profile images for clients and vendors
- **Vendor Management**: Complete vendor module with image support
- **Services**: Add services to invoices and estimates alongside items
- **Invoice & Estimate Management**: Create, edit, send, download PDFs
- **Client Management**: Full client database with document uploads
- **Dashboard**: Revenue overview with charts and statistics
- **Month Filtering**: Filter invoices/estimates by month
- **CSV Export**: Export data to CSV format

## 🛠 Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui components
- **State Management**: React Context API
- **Internationalization**: react-i18next
- **Charts**: Recharts
- **Icons**: Lucide React

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/systemapropos-sp/invoicepro-saas.git

# Navigate to project
cd invoicepro-saas

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
